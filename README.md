# mm-eyebleach
Get your daily dose of cute animals pics without leaving Mattermost

<img src="https://i.redd.it/scnj8h0v7kf71.jpg" alt="pic" width="380" height="380">

## Usage

1. Add a Mattermost slash command (e.g. `/eyebleach`) and point it at `http://yourserver/image`.
2. Run `npm install && node main.js`
3. Type `/eyebleach` in Mattermost
4. Profit
